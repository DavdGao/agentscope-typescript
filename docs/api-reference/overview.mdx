## Updating Documentation

The API documentation is automatically generated from source code. To update:

```bash
cd packages/agentscope
pnpm docs:generate
```

This will regenerate all API documentation based on the latest code and JSDoc comments.

## Contributing

When adding new features or modifying APIs:

1. Add comprehensive JSDoc comments to your code
2. Include `@param` tags for all parameters
3. Include `@returns` tags for return values
4. Add usage examples in JSDoc when helpful
5. Run `pnpm docs:generate` to update the API docs
