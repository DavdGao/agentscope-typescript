## Overview

Friday is built with a modern Electron architecture separating concerns between main process, renderer process, and preload scripts.

## Architecture Diagram

```mermaid
graph TB
    A[Renderer Process - React] --> B[Preload Scripts]
    B --> C[Main Process - Electron]
    C --> D[AgentScope Core]
    D --> E[Language Models]
    D --> F[Tools & Skills]
    D --> G[MCP Servers]
```

## Components

### Main Process

Handles:

- Agent lifecycle management
- IPC communication
- File system operations
- Database management

### Renderer Process

Handles:

- UI rendering with React
- User interactions
- Message display
- Real-time updates

### Preload Scripts

Provides secure bridge between renderer and main process.
