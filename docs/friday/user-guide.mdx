## Getting Started

After launching Friday, you'll see the main chat interface.

## Basic Usage

### Starting a Conversation

1. Type your message in the input box
2. Press Enter or click Send
3. Friday will respond with streaming text

### Using Tools

Friday can use various tools:

```
You: What files are in my current directory?

Friday: Let me check that for you.
[Calls Bash tool: ls -la]
Here are the files in your directory:
- file1.txt
- file2.js
...
```

### Managing Sessions

- **New Session**: Click the "New Chat" button
- **Switch Sessions**: Click on a session in the sidebar
- **Rename Session**: Right-click and select "Rename"
- **Delete Session**: Right-click and select "Delete"

## Advanced Features

### MCP Integration

Friday supports Model Context Protocol for extended capabilities.

### Custom Skills

Add custom skills to extend Friday's capabilities.

## Keyboard Shortcuts

- `Cmd/Ctrl + N`: New session
- `Cmd/Ctrl + K`: Focus search
- `Enter`: Send message
- `Shift + Enter`: New line
