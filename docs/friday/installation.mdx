## Build from Source

### Prerequisites

- Node.js 18.0.0+
- pnpm 8.0.0+

### Clone and Install

```bash
git clone https://github.com/agentscope-ai/agentscope-typescript.git
cd agentscope-typescript
pnpm install
```

### Run in Development

```bash
cd apps/friday
pnpm dev
```

### Build for Production

```bash
cd apps/friday
pnpm build
```

## Configuration

Create a configuration file at `~/.friday/config.json`:

```json
{
    "models": {
        "gpt-4": {
            "provider": "openai",
            "apiKey": "your-api-key",
            "model": "gpt-4o"
        }
    }
}
```
