## Overview

AgentScope uses an event-driven architecture to track every step of agent execution. Events enable real-time streaming, progress tracking, debugging, and frontend integration.

## Event Categories

### Lifecycle Events

- `RUN_STARTED` - Agent execution begins
- `RUN_FINISHED` - Agent execution completes
- `EXCEED_MAX_ITERS` - Maximum iterations reached

### Model Events

- `MODEL_CALL_STARTED` - Model API call begins
- `MODEL_CALL_ENDED` - Model API call completes with token usage

### Content Events

- Text: `TEXT_BLOCK_START`, `TEXT_BLOCK_DELTA`, `TEXT_BLOCK_END`
- Thinking: `THINKING_BLOCK_START`, `THINKING_BLOCK_DELTA`, `THINKING_BLOCK_END`
- Binary: `BINARY_BLOCK_START`, `BINARY_BLOCK_DELTA`, `BINARY_BLOCK_END`

### Tool Events

- Tool Call: `TOOL_CALL_START`, `TOOL_CALL_DELTA`, `TOOL_CALL_END`
- Tool Result: `TOOL_RESULT_START`, `TOOL_RESULT_TEXT_DELTA`, `TOOL_RESULT_BINARY_DELTA`, `TOOL_RESULT_END`

### Human-in-the-Loop Events

- `REQUIRE_USER_CONFIRM` - User confirmation needed
- `REQUIRE_EXTERNAL_EXECUTION` - External execution needed
- `USER_CONFIRM_RESULT` - User confirmation result
- `EXTERNAL_EXECUTION_RESULT` - External execution result

## Event Interfaces

### Base Event

All events extend this base interface:

```typescript
interface EventBase {
    id: string; // Unique event ID
    createdAt: string; // ISO timestamp
}
```

### RUN_STARTED

```typescript
interface RunStartedEvent extends EventBase {
    type: 'RUN_STARTED';
    sessionId: string;
    replyId: string;
    name: string; // Agent name
    role: 'user' | 'assistant' | 'system';
}
```

## Usage Example

```typescript
import { EventType } from '@agentscope-ai/agentscope/event';

for await (const event of agent.replyStream({ msgs: userMessage })) {
    switch (event.type) {
        case EventType.RUN_STARTED:
            console.log(`Agent ${event.name} started`);
            break;

        case EventType.MODEL_CALL_STARTED:
            console.log(`Calling model: ${event.modelName}`);
            break;

        case EventType.TEXT_BLOCK_DELTA:
            process.stdout.write(event.delta);
            break;

        case EventType.TOOL_CALL_START:
            console.log(`\nCalling tool: ${event.toolCallName}`);
            break;

        case EventType.TOOL_RESULT_TEXT_DELTA:
            console.log(`Tool output: ${event.delta}`);
            break;

        case EventType.MODEL_CALL_ENDED:
            console.log(`\nTokens: ${event.inputTokens} in, ${event.outputTokens} out`);
            break;

        case EventType.RUN_FINISHED:
            console.log('Agent finished');
            break;
    }
}
```
